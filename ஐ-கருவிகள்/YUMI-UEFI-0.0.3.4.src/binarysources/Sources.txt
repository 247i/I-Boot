This file outlines where to find the third party tools (tools authored entirely by others) that are either used in UEFI YUMI or used to create it.
The binarysources.zip file contains the sources for grub, Syslinux, 7zip, dd, and Fat32format. 
It can be found at http://www.pendrivelinux.com/downloads/UEFI-YUMI/binarysources.zip

NSIS (Nullsoft Scriptable Install System) was used to create YUMI

Homepage: http://nsis.sourceforge.net/Main_Page
Binary/Source: http://nsis.sourceforge.net/Download


core.img, lnxboot.img, BOOTX64.efi, i386-pc and x86_64-efi modules (GRUB2)

Homepage: http://sourceforge.net/projects/grub4dos/files/
Source: http://alpha.gnu.org/gnu/grub/grub-2.02~beta3.tar.gz
Binary: Binaries compiled from source using Ubuntu
